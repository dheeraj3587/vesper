## 🏗️ **PROJECT STRUCTURE** 🏗️

```
vesper/
├── 🧠 src/           # Core source code
│   ├── main.cpp     # Main driver
│   ├── Lexer.cpp    # Lexical analyzer
│   └── Parser.cpp   # Parser implementation
├── 📋 include/      # Header files
│   ├── Lexer.h      # Lexer interface
│   ├── Parser.h     # Parser interface
```
